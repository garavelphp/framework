## Run the computer

Clone the project

```bash
  git clone https://github.com/garavelphp/freamwork.git
```

Go to the project directory

```bash
  cd freamwork
```

Run with docker

```bash
  docker compose up -d --build
  docker exec -it garavelphp-web-1 bash
```

Load the dependencies

```bash
  composer install
```
